インストール

準備

この ソフトウェアを利用するためには、Java SE 6 以上の実行環境(JRE)が必要です。
配布パッケージに含まれていない場合には、以下のURLからダウンロードしてください。

http://www.java.com/ja/download/manual.jsp

配 布パッケージ内に jre-XXXXXX.exe があった場合、それが実行環境です。

インストール手順

  1. まず、Java実行環境のインストーラファイルを開き、コンピュータにインストールします。
  2. Javaのインストールが終了すると、配布パッケージに含まれている install.jar が実行可能 になっているはずですので、ダブルクリックで実行します。ターミ ナルから起動する場合には、
    java -jar install.jar
    として実行します。
  3. インストーラのウィンドウが開きます。インストールディレクトリを選んで｢次へ｣を押 し、「終了」ボタンが表示されたら、それを押してインストールを終わります。

注意点

現在 のバージョンでは、スタートメニューやデスクトップにアイコンを追加しません。必要でしたら、インストール先のディレクトリにある EJAdvisor.bat へのショートカットをデスクトップに追加してください。
